# Flint Ionic Dashboard

Agent task dashboard scaffold built with Ionic 8 + Angular 20 + Capacitor 8.

## Quick Start

```bash
npm install
npm run start
```

## Current Status

- Phase 1 scaffold complete (shell pages + stubs)
- Routes:
  - `/dashboard`
  - `/task/:id`
  - `/new-thread`
- Services (stubbed):
  - `TaskService`
  - `SocketService`
  - `ThreadService`

## Build + Lint

```bash
npm run lint
npm run build
```

## Repo-local MCP

This repo includes official Angular CLI MCP configuration at:
`/.vscode/mcp.json`

After MCP config changes, reload VS Code or start a new chat session to refresh visible tools.
